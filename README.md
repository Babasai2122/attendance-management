# Attendance Management System
Attendance Management System implements the 'punch in / punch out' based concept of attendance management.

### Employee

- Punch In time /  Punch Out time record
- Hours spent record
- Attendance sheet
- Hours/day graph

### Admin

- User Management
- Attendance sheet of user
- Hours/month graph
- Hours/day graph
- Update Attendance
- Set leaves
- Set holidays

## TECHNOLOGIES ##

- Laravel

- Bootstrap

- MySql

- Apache Server
